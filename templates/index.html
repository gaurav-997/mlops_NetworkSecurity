About This Application

🎯 Purpose

This application uses machine learning to detect phishing URLs by analyzing 30 different features extracted from URLs and websites. The model is trained on a comprehensive dataset of legitimate and phishing websites.

🏗️ Architecture

  • Data Ingestion: Load and split data into train/test sets
  • Data Validation: Validate schema and detect data drift
  • Data Transformation: KNN imputation and preprocessing
  • Model Training: Train multiple classifiers, select best model
  • Model Evaluation: Compare with production model
  • Model Registry: Track experiments with MLflow

🔧 Technology Stack

🐍 Backend:
FastAPI, Python
🤖 ML:
Scikit-learn, Pandas
📊 MLOps:
MLflow, DVC
🎨 Frontend:
HTML, CSS, JavaScript

📖 API Endpoints

  • POST /train - Trigger model training pipeline
  • POST /predict - Single URL prediction
  • POST /predict_csv - Batch prediction from CSV
  • GET /status - Check API and model status
  • GET /docs - Interactive API documentation
